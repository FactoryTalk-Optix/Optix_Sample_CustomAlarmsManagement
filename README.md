# Custom Alarms Management

This sample project shows how to create a 100% custom alarms management system.

This could be useful on projects where a giant amount of alarms is being processed and there is no need to use the native functionalities of Optix alarms (like exposing them via OPCUA) and helps saving memory and CPU time as alarms are handled as database rows.
